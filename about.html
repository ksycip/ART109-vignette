About Vignette

Vignette emulates a multipurpose sketch tool that is capable of curing oneâ€™s boredom when sitting at their desk. Consider this your virtual desk tool away from home! The monochromatic color scheme was carefully chosen so that the site conveys a calming presence.
On Vignette you have a vast array of activities, the user is encouraged to choose among the options in the top menu bar. Embrace the freedom to color some Van Gogh coloring pages, take notes, sketch something fun, or create an interesting pattern by connecting the dots! On all of these pages you can draw and choose a color with the ColorPicker on the top right corner and click the 'space' bar to clear the screen after drawing if you wish to restart. Make sure to screenshot your work so you can document your boredom artwork!
This site primarily uses HTML and CSS, and contains minor JavaScript coding languages. CSS animation and pattern code was heavily relied on for achieving the outcome of this project.
